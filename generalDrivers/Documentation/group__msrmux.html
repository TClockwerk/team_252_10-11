void MSRXMUXsetupChan ( tSensors  link,
byte  chan,
TSensorTypes  chantype,
TSensorModes  chanmode,
byte  delay 
)

Send a direct command to the PFMate sensor

Note: this is an internal command and should not be called directly.

Parameters:
link the PFMate port number
chan the channel to be configured
chantype the sensor type connected to the channel
chanmode the sensor mode of the sensor
delay the initialisation delay time
Returns:
true if no error occured, false if it did
