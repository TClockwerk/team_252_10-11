bool NXTServoSetPos ( tSensors  link,
ubyte  servochan,
int  position 
)

Tell the servo to move to the specified position. It will use whatever speed is already set in the speed register. Use NXTServoSetSpeed() to change this setting or use NXTServoSetPos() to set both at once.

The position argument is clipped at 500 2500 to prevent damage to the servo.

Parameters:
link the NXTServo port number
servochan the servo channel to use
position the position to move the servo to
Returns:
true if no error occured, false if it did
