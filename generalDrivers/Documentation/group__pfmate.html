bool _MSPFMcontrolMotors ( tSensors  link,
byte  chan,
byte  mselect,
byte  motorA_op,
byte  motorA_sp,
byte  motorB_op,
byte  motorB_sp 
)

Control motors A and B with the PFMate. Must be followed by a MSHID_GOCMD command using _MSPFMsendCommand()

Note: this is an internal command and should not be called directly.

Parameters:
link the PFMate port number
chan the channel of the IR receiver, value of 1-4
mselect select the motors to operate on, 0: both motors, 1: motor A, 2: motor B
motorA_op motor A operation, 0: float, 1: forward, 2: reverse, 3: brake
motorA_sp the speed at which motor A should turn, value between 1-7
motorB_op motor B operation, 0: float, 1: forward, 2: reverse, 3: brake
motorB_sp the speed at which motor B should turn, value between 1-7
Returns:
true if no error occured, false if it did
