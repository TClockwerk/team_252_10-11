ubyte arr [MAX_ARR_SIZE]

Detailed Description

Array of bytes as a struct, this is a work around for RobotC's inability to pass an array to a function. The int has to be there or it won't work.

Examples:

EEPROM-test1.c, MSLL-test2.c, and MSLL-test3.c.
